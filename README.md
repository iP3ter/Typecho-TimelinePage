# TimelinePage for Typecho

在 Typecho 文章或页面中写 `[timeline]...[/timeline]`，可自动渲染为时间轴样式。

## 安装

1. 将 `TimelinePage` 文件夹放到 Typecho 的 `usr/plugins/` 目录。
2. 进入后台 `控制台 -> 插件`。
3. 启用 `TimelinePage`。

## 使用示例

### 1) 仅日期 + 文本

```text
[timeline]
2026-03-01|项目启动
2026-03-03|完成设计稿
2026-03-06|发布首个版本
[/timeline]
```

### 2) 加入标题

```text
[timeline]
2026-03-06|更新|版本升级与修复|https://example.com/img/a.jpg
2026-03-10|发布|上线新页面|https://example.com/img/b.jpg, https://example.com/img/c.png
[/timeline]
```

说明：`标题`（第 2 段）当前不显示；若第 3 段为空，会自动将第 2 段作为正文显示。

## 语法规则

- 每行一条记录，支持：
- `日期|描述`
- `日期|标题|描述|图片URL`
- `图片URL` 可省略；多图使用英文逗号分隔。
- 以 `#` 开头的行会被忽略（可用于注释）。
- 描述支持行内 Markdown：`**加粗**`、`*斜体*`、`~~删除线~~`、`` `代码` ``、`==高亮==`、`[链接](https://...)`、`![图片](https://...)`。
- 兼容编辑器输出的 HTML 链接和图片：`<a href="...">`、`<img src="...">`。

## 样式说明

- 正文内容字体已调为更粗（`font-weight: 500`）。
- 日期、圆点、竖线颜色默认自动跟随主题主色。

## 插件设置（日期颜色）

进入后台 `控制台 -> 插件 -> TimelinePage -> 设置`，可看到两个配置项：

- `自定义日期颜色（默认模式）`
- `自定义日期颜色（夜间模式）`

填写规则：

- 留空：自动跟随主题主色（推荐）。
- 填写：使用你指定的颜色。
- 仅支持 Hex 格式，例如 `#10bfa8` 或 `10bfa8`。
- 若只填了默认模式颜色，夜间模式会沿用该颜色。

## 备注

- 插件对文章和页面正文都生效。
- 图片支持点击弹层预览（支持 `ESC` 或点击空白关闭）。
